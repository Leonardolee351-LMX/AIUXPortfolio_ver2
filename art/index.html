艺术作品 / New Media Works

从《数媒方向作品集》拆分出的四个主作品，以图集形式独立呈现。上下滚动浏览，点击图片可放大预览。

Project 01

Looming Dreamer

公共空间交互装置，以雅卡尔织机机制为灵感，编织中法文化记忆与人际连接。
Project 02

Luminous & Silhouette

视听互动 VJ 演出项目，融合古筝与电子音色的张力，探索传统与现代的共振关系。
Project 03

FingerTip praying

从祈福 meme 与手势仪式切入的互动装置研究，关注压力释放、社交传播与情绪共鸣。
Project 04

Surreal Wanderer

基于《Sonny Boy》漂流叙事的空间翻译实验，通过图表与符号推演探索空间叙事结构。
